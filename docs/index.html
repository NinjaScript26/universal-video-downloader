Quick start

# 1. get yt-dlp (once)
pipx install yt-dlp        # or see README for the standalone binary

# 2. get the app
git clone https://github.com/NinjaScript26/universal-video-downloader.git
cd universal-video-downloader

# 3. run it
python3 app.py             # then open http://127.0.0.1:8000

Also install ffmpeg (for MP3 / best-quality merging) and, for YouTube, have node or deno on your PATH — auto-detected.

Features

  • Two modes — one video / short / reel, or an entire channel, playlist, shorts tab or reels feed (optional "max videos" limit)
  • Quality picker: best / 1080p / 720p / 480p / MP3 (audio only)
  • Private or login-gated content: reuses your own browser's logged-in session — no cookie files to export
  • Live progress streamed straight from yt-dlp, with cancel
  • Zero Python dependencies — one file of stdlib, plus the yt-dlp CLI
